# 💰 Smart Spend Tracker

A modern and responsive expense tracking web application built with **HTML**, **CSS**, and **Vanilla JavaScript**. It helps users manage their budget, record expenses, monitor spending, and view remaining balance through an intuitive dashboard.

## 🚀 Features

- 📊 Dashboard displaying:
  - Total Budget
  - Total Expenses
  - Remaining Balance
- 💵 Set and update your budget
- ➕ Add new expense transactions
- ✏️ Edit existing expenses
- 🗑️ Delete expense records
- 🏷️ Categorize expenses
- 🔍 Filter expenses by category
- 💾 Data persistence using Local Storage
- 📱 Responsive design for desktop and mobile devices
- 🎨 Clean and modern UI with Font Awesome icons

## 📸 Application Overview

### Dashboard
- Displays current budget
- Shows total expenses
- Calculates remaining balance automatically

### Budget Management
- Set your total budget.
- Warns if the new budget is lower than current expenses.

### Expense Management
Each expense includes:
- Title
- Amount
- Category

Supported categories:
- Food / Catering
- API & Software
- Hardware / Gadgets
- Marketing & Swag
- Other

### Expense Operations

- Add new expense
- Update existing expense
- Delete expense
- Filter by category

---

## 💾 Data Storage

The application stores data in the browser using **Local Storage**, including:

- Budget
- Expense history

Data remains available after refreshing or reopening the browser.
